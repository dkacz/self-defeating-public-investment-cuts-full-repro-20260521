# Frozen Adopted Model Inputs

This folder contains the small model-input slice used by the adopted manuscript specification.

The files are copied from the locally frozen source artifact so the public package can rebuild its tables without reading paths outside this directory. These are not live downloads and are not FIGARO diagnostics; they are the official-TiVA GFCF, real-PPP and net-worth model inputs used by the manuscript tables.

Hashes are written to `data/provenance/frozen_inputs_manifest_20260514.csv` by `code/run_public_repro.py`.
